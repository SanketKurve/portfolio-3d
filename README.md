## ðŸ› ï¸ Technical Architecture

This portfolio is engineered for performance and scalability.

*   **Core Framework**: [React 18](https://reactjs.org/)
*   **Build Tool**: [Vite](https://vitejs.dev/) (for lightning-fast HMR and bundling)
*   **3D Engine**: [Three.js](https://threejs.org/) via `@react-three/fiber` & `@react-three/drei`
*   **Animations**: [Framer Motion](https://www.framer.com/motion/) (UI transitions) & Custom CSS Keyframes
*   **Styling**: Pure CSS3 with extensive interactive Custom Properties (Variables)
*   **Optimization**:
    *   `requestAnimationFrame` for custom cursor logic.
    *   Geometry simplification for 3D assets to ensure 60fps on mobile.
    *   Lazy loading key components.
